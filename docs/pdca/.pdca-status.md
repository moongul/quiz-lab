## Usage

### JSON Status File

상태 추적은 `.pdca-status.json`에서 관리됩니다:

```json
{
  "activeFeatures": [
    {
      "name": "feature-name",
      "phase": "plan|design|do|check|act|report",
      "matchRate": 0-100,
      "iteration": 0-5
    }
  ]
}
```

### Commands (Natural Language)

```
"새 기능 {feature} 계획해줘"    → Plan 문서 생성
"설계 문서 작성해줘"            → Design 문서 생성  
"구현 시작해줘"                 → Implementation 가이드
"갭 분석해줘"                   → Gap Analysis (Check phase)
"자동 수정해줘"                 → Auto-fix iteration (Act phase)
"완료 보고서 작성해줘"          → Completion Report
"상태 확인해줘"                 → Current status
"다음 단계 알려줘"              → Next step guide
```

### Phase Flow

```
Plan → Design → Do → Check → (Act) → Report → Archive
                       ↓
                   < 90%? → Iterate (max 5x)
```
